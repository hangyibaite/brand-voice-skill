# Anti-AI Universal Banned List

Apply to every category in every generated skill. The personalized skill must copy this file verbatim into its own `assets/anti-ai.md` and load it on every write.

## Banned phrases

- Em-dashes (—) — use commas, periods, or parentheses instead
- "Delve into" / "delve"
- "Navigate the complexities"
- "In today's fast-paced world"
- "Unlock" (as in "unlock your potential")
- "Leverage" as a verb
- "Robust"
- "Seamless"
- "Comprehensive"
- "Cutting-edge"
- "Game-changer" / "game-changing"
- "Tapestry"
- "Underscore" / "underscores"
- "It's worth noting that"
- "When it comes to X..."
- "At the end of the day" (as filler)
- "Moving forward"
- "Furthermore" / "Moreover" / "Additionally" as paragraph openers

## Banned rhetorical moves

- Tricolon parallelism as a default rhythm ("X, Y, and Z" repeatedly across sentences)
- "Not just X — but Y" / "It's not just about X, it's about Y"
- Hedging stacks ("It can be argued that, in many ways, X may sometimes...")
- Three-item lists as a default structure when prose would work
- Summary sentences that restate the previous paragraph
- "In conclusion" or any paraphrase of it
- Generic transition adverbs at paragraph starts (However, Furthermore, Moreover)
- Tone-neutral sales language ("This solution offers...", "Our platform provides...")
- Asking rhetorical questions and immediately answering them in the next sentence as a default device

## Banned formatting (unless user samples show otherwise)

- Excessive bolding inside prose
- Headers in short-form content
- Generic bullet lists when prose works
- Emoji unless samples show emoji usage
- Numbered lists for everything

## Override clause

If the user's actual samples show heavy use of something on this list, the user's pattern wins. Note the exception in the relevant category file. Default is: this list overrides.
